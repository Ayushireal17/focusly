# Focusly / Focus Timer App
A simple, clean, and modern  Timer web application designed to improve focus and productivity using the Pomodoro Technique. Built with HTML, CSS, Bootstrap, and JavaScript, this app provides an intuitive interface with essential features for effective time management.

# Features
Start, Pause, and Reset timer controls
Work and Break modes
Fully responsive design (mobile-friendly)

# Technologies Used
HTML5
CSS3
Bootstrap 5
JavaScript

# Purpose
This project demonstrates front-end development skills including DOM manipulation, timer logic, responsive UI design, and user interaction handling.

# How to Use
Start the timer to begin a focus session
Work until the timer ends
Take a break when prompted
Track completed sessions
Repeat for maximum productivity
